Summary

The purpose of this project was to analyze how Latitude can impact four specific elements of the weather. To accomplish this, we used the OpenWeatherMap API to gather data on more than 500 random cities.

After collecting real-time weather data for each city, we used scatter plots to visualize the relationship between the Latitude of each city and the day's maximum temperature, current humidity, cloudiness, and wind speed.
